# Your Portfolio Images

This folder contains all images for your portfolio.

## 📸 Image Organization

Add your images here in the following structure:

```
images/
├── profile.jpg           # Your headshot (400x400px)
├── project-1.jpg         # First project (800x600px)
├── project-2.jpg         # Second project (800x600px)
├── project-3.jpg         # Third project (800x600px)
└── project-4.jpg         # Fourth project (800x600px)
```

## 📷 How to Reference Images in Code

In your components, reference images like this:

```jsx
// In ProjectGallery.jsx or other components:
image: '/images/project-1.jpg'

// The leading slash (/) means it comes from the public folder
```

## 💡 Tips

- **Optimize first**: Use [TinyPNG](https://tinypng.com) or [Squoosh](https://squoosh.app) before uploading
- **Consistent sizes**: Keep project images at 800x600px
- **File formats**: JPG for photos, PNG for graphics
- **File names**: Use lowercase, hyphens for spaces
